Apple working on a fix to glitch that lets pranksters crash iPhones

1A newly-discovered glitch in Apple's software can cause iPhones to mysteriously shut down when they receive a certain text message.
2Apple says it's aware of the problem and is working on a fix.
3But some pranksters are sharing information about the glitch on social media and using it to crash other peoples' phones.
4The problem only occurs when the iPhone receives a message with a specific string of characters, including some Arabic characters, according to several tech blogs.
5When an iPhone isn't being used, it typically shows a shortened version of the message on the phone's lock screen.
6That shortened combination of characters apparently triggers the crash.
7Affected phones will restart automatically.
8Owners can prevent the problem by using phone settings to turn off the message "previews" feature.
