About time - Microsoft to ditch Internet Explorer brand

1Microsoft is finally ditching Internet Explorer.
2The software maker has confirmed that it will use a new name for its upcoming Windows 10 browser, The Verge reports.
3But it hasn't decided on one yet - it's currently codenamed Project Spartan.
4For years, Internet Explorer has been the browser the world loves to hate.
5After a series of security flaws and holes in the early 2000s, it developed a serious image problem and never fully recovered.
6"We're now researching what the new brand, or the new name, for our browser should be in Windows 10," Chris Capossela said at Microsoft's Convergence event.
7"We'll continue to have Internet Explorer, but we will also have a new browser called Project Spartan, which is code-named Project Spartan.
8"We have to name the thing."
